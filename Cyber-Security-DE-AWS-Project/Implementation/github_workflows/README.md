# GitHub Actions

This folder contains sample GitHub Actions workflows demonstrating how infrastructure and AWS Glue resources can be deployed using Infrastructure as Code (IaC).

## Workflow Overview

The deployment workflow performs the following activities:

1. Trigger deployment on code changes.
2. Validate CloudFormation templates.
3. Upload Glue ETL scripts to Amazon S3.
4. Deploy or update AWS resources using CloudFormation.
5. Verify successful deployment.

## Highlights

- CI/CD automation
- Infrastructure as Code
- Automated Glue deployments
- CloudFormation validation
- Environment-based deployment
